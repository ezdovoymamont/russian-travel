## Описание проекта
Данный учебный проект представляет типовой одностраничный сайт.
Проект посвящен путешествию по России и создан в рамках прохождения 3 спринта курса Веб-разработчик.

## Стек технологий:
- HTML5;
- CSS3:
  - Flexbox;
  - Grid Layout;
  - Positioning;
  - Adaptive UI;
  - Media Queries;
- Методология БЭМ;
- Фйловая структура Nested БЭМ;
- Вёрстка по макету в Figma.
